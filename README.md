# 🤖 AskBuddy - AI QnA Bot

## 🚀 About Project
AskBuddy is an AI-powered question-answering chatbot built using LangChain and Google Gemini. It provides fast and intelligent responses through a simple Streamlit interface.

## 🛠️ Tech Stack
- Python
- LangChain
- Google Gemini
- Streamlit

## ✨ Features
- AI-based QnA system
- Fast responses
- Simple UI
- Easy to use

## ⚙️ Setup Instructions

1. Clone the repository:
git clone https://github.com/mdSohail07/AskBuddy.git

2. Install dependencies:
pip install -r requirements.txt

3. Run the app:
streamlit run 1_qna_bot.py

## 🔑 Environment Variables

Create a `.env` file and add your API key:

GOOGLE_API_KEY=your_api_key_here
